## Project Overview
This project is an online clothes renting platform that allows users to rent fashionable clothing for short-term use instead of purchasing them. The platform focuses on affordability, sustainability, and convenience by enabling users to access a wide range of outfits for occasions such as parties, weddings, and formal events.

## Problem It Solves
Buying clothes for occasional use is expensive, space-consuming, and environmentally unsustainable. Many users wear such clothes only once or twice. This platform reduces unnecessary purchases by providing a rental-based alternative that saves money and promotes sustainable fashion.

## Target Users (Personas)
- College students looking for affordable fashion options  
- Young professionals renting outfits for events  
- Users interested in sustainable and eco-friendly fashion  
- Fashion-conscious individuals who prefer variety without ownership  

## Vision Statement
To create a reliable and user-friendly clothing rental platform that makes fashion accessible, affordable, and sustainable for everyone.

## Key Features / Goals
- User registration and secure login  
- Browse and search rental clothing  
- Clothing categorization by type and occasion  
- Rental booking and return management  
- Admin management of inventory and rentals  

## Success Metrics
- Number of active users  
- Successful rental transactions  
- User satisfaction and repeat usage  
- Reduction in unnecessary clothing purchases  

## Assumptions & Constraints
- Users have internet access and basic digital literacy  
- Initial inventory is limited  
- The system is developed within academic timelines  
- Online payments may be simulated or basic in initial versions  

## Branching Strategy
This project follows GitHub Flow.
The main branch contains stable code.
Feature development is done in separate feature branches which are later merged into main.

## Tech Stack
